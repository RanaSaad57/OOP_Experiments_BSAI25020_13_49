1. MODULES COMPLETED
--------------------

Scientific Calculator
- Completed arithmetic and scientific operations.
- Supports addition, subtraction, multiplication, division, powers, square,
  square root, trigonometric functions, and logarithms.
- Implemented as a reusable UI prefab.

Data Recording Notebook
- Records experiment name, observations, and time.
- Supports long observations and multiple pages.
- Implemented as a reusable UI prefab.

Measurement Reading UI
- Displays readings for common laboratory instrument types.
- Supports units, precision, previous/next instrument selection, reset, and a
  demonstration mode.
- External student devices must call the public reading API during integration.

Graph Plotting System
- Adds manual data points and renders a line graph.
- Calculates slope.
- Displays major and minor grid divisions and axis scale information.
- Implemented as a reusable UI prefab.

Weighing Balance
- Includes a 3D digital balance and a toggleable control panel.
- Displays sample name, grams, and kilograms.
- Supports simulation, tare, reset, and close controls.
- Final integration with external draggable samples depends on the common game
  interaction system selected by the class.

2. MODULE LIMITATIONS OR FUTURE INTEGRATION
-------------------------------------------

- Measurement Reading UI includes a demonstration data source because the final
  interfaces of voltmeters, thermometers, and other student-made instruments
  were not defined when the module was developed.
- Weighing Balance includes simulation controls. Automatic mass detection from
  arbitrary external objects requires those objects to expose mass through an
  agreed class interface or Rigidbody.
- The modules are prefabs and can be integrated, but visual positioning may
  need adjustment in another student's scene or canvas.

4. PRACTICAL 4 - INCLINED PLANE
-------------------------------

Completed:
- Experiment information, apparatus, theory, and procedure screen.
- Guided Practice and Independent Mode.
- Toggleable instructions and highlighted workflow guidance.
- Adjustable angle, friction, and target distance with sliders, input fields,
  plus buttons, and minus buttons.
- Clickable 3D inclined-plane apparatus and closeable controls.
- Rolling-ball simulation, reset, sampling, timing, and distance measurement.
- Automatic data notebook containing s, t, 2s, t^2, and acceleration.
- Dynamic graph of 2s against t^2 with major/minor boxes, labels, manual data
  entry, and slope calculation.
- Formula calculator and manual scientific calculator.

Remaining limitations:
- Results are an educational Unity physics simulation and can vary slightly
  with the physics timestep.
- Real laboratory imperfections such as air resistance and instrument reaction
  time are simplified.
